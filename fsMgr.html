fsMgr¶

This module acts as Front End of fsMgr and provides searching capabilities to users

class fsMgr.Search(application, request, **kwargs)[source]¶

This class provides top 10 search results for a keyword.

It indexes the root tree, searches the documents and renders the search results with url and runtime

initialize() defaults search query param to None.

post() handles the search paramter sent in post request.

  • parameters, types, return and return types:

    :return: top 10 search results for the query
class fsMgr.AdvSearch(application, request, **kwargs)[source]¶

AdvSearch class provides the user the capability of advanced search results for a keyword.

AdvSearch involves didyoumean, morelikethis and highlighted search

get() handles the get request.

  • parameters, types, return and return types:

    :return: advanced search form to the user
class fsMgr.Highlighted(application, request, **kwargs)[source]¶

Highlighted class provides highlighted search results for a keyword.

It searches the documents for keywords and renders the highlighted search results with url and contents

post() handles the search paramter sent in post request.

  • parameters, types, return and return types:

    :return: highlighted search results for the query
class fsMgr.MoreLikeThis(application, request, **kwargs)[source]¶

MoreLikeThis class provides document based search results.

Contents of a given document are searched and documents that have similar contents would be return as result

post() handles the document path/url sent in post request.

  • parameters, types, return and return types:

    :return: morelikethis document results for the query
class fsMgr.DidYouMean(application, request, **kwargs)[source]¶

DidYouMean class provides search results based on spell check.

For instance, if someone searches for ‘pyton’, it has capability of understanding it as ‘python’ and searching for it. or ‘hart’ is treated as ‘heart’

post() handles the search query sent in post request.

  • parameters, types, return and return types:

    :return: search results for the spell checked query
class fsMgr.FS(application, request, **kwargs)[source]¶

FS class reads through files of filesystem located under root/ and helps to display the file contents during search results or when user requests to view file

get() handles the root/ path sent.

  • parameters, types, return and return types::

    return:file contents based on request

_async_callback() gets file meta information

  • parameters, types, return and return types::
    parameter:items, list of files in tree
    type:list
    return:gathers file meta information
